# Task2_EDA_titanic_dataset
Performed Exploratory Data Analysis (EDA) on the Titanic dataset as part of the CodeAlpha Data Analytics Internship. Analyzed passenger demographics, survival patterns, and feature relationships using Python, Pandas, Matplotlib, and Seaborn. Includes data cleaning, visualization, insights, and detailed project documentation.
Titanic Dataset - Exploratory Data Analysis (EDA)

 📌 Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship. The objective is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, and factors that influenced passenger survival.

🎯 Objectives

* Understand the dataset structure and features.
* Perform data cleaning and preprocessing.
* Analyze passenger demographics.
* Explore survival patterns.
* Visualize key insights using charts and graphs.

 📂 Dataset Information

The dataset contains information about Titanic passengers, including:

* Passenger ID
* Age
* Gender
* Passenger Class
* Fare
* Embarkation Port
* Survival Status

🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

📊 Exploratory Data Analysis

The following analyses were performed:

* Data Cleaning and Missing Value Handling
* Descriptive Statistics
* Survival Rate Analysis
* Gender-wise Survival Comparison
* Passenger Class Analysis
* Age Distribution Analysis
* Fare Distribution Analysis
* Correlation Analysis
* Data Visualization

 🔍 Key Insights

* Female passengers had a higher survival rate than male passengers.
* First-class passengers were more likely to survive.
* Younger passengers showed relatively better survival chances.
* Passenger class and gender significantly influenced survival outcomes.
